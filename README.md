# skiren-site

Promo landing page for **Skiren** — the adaptive, lossless layer that reshapes AI output into its
lowest-cognitive-effort form. Sibling to [**Hearth**](https://trollyal.github.io/hearth-site/) in a
shared Nordic-mythology brand family (warm Hearth / cold Skiren).

**Live:** https://trollyal.github.io/skiren-site/

## What it is

A single self-contained `index.html` — no build step, no dependencies. Deployed via GitHub Pages from
`main` at the repo root (`.nojekyll` disables Jekyll). Mirrors the structure and motion language of the
Hearth site, swapped to Skiren's **cold spectrum** (rime `#5FA0D6` accent, aurora `#57C7AE` signature,
one amber caution note) and the hexagon/Ansuz mark.

Highlights:

- **Live reshape centerpiece** — watch a wall of AI prose route through the precision-first cascade
  (structural scan → semantic classifier → reshape → lossless check) and become a comparison table with
  a recommended column, respecting `prefers-reduced-motion`.
- The six locked **modes** (Prose · Steps · Key-Points · Comparison · Structured-Reference · Diagram),
  the **renderer** surface, the **lossless** promise, the engine/protocol features, and the three
  product layers (Protocol · Composer · Renderer).
- A **family band** linking Skiren ↔ Hearth.

```bash
python3 -m http.server 8000   # → http://localhost:8000
```
